Simplified Annotations

This inspection evaluates string expressions marked with the @ResourcePath annotation to check if resource files exist.

Supported String Resolution

  • String literals
  • Binary concatenation (e.g., "a" + "b")
  • Constants and field initializers
  • Simple methods with direct String return values
  • Enum constants and their constructor parameters

Code Example

    // Class Field
    @ResourcePath
    private String configFile = "config/settings.xml";

    // Class Field (with base)
    @ResourcePath(base = "images")
    private String iconFile = "icon.png"; // images/icon.png

    // Method Parameter (with base)
    public void loadResource(@ResourcePath(base = "templates") String fileName) {
        // implementation
    }

    // Enum Constructor
    public enum Status {
        OK("logo.png"); // assets/logo.png

        private final String iconPath;

        Status(@ResourcePath(base = "assets") String iconPath) {
            this.iconPath = iconPath;
        }
    }
